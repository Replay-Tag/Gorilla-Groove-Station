# Gorilla-Groove-Station
A Gorilla Tag Mod that implements a Record player into the game
# Music
* Monke Need to Swing
* Cave Wave
* Campfire
* Nice Gorilla Store
* What the Future Holds
# Features
* Record Player With working music
* Skip songs
* Pause songs
